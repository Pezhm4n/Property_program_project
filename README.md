# سیستم مدیریت املاک (Property Management System)

سیستم مدیریت املاک یک برنامه جامع برای مدیریت، جستجو و گزارش‌گیری از املاک مسکونی، تجاری و زمین است. این برنامه با واسط کاربری پایتون و هسته C توسعه داده شده است.

## ویژگی‌ها

- ثبت و مدیریت اطلاعات املاک مسکونی، تجاری و زمین
- جستجوی پیشرفته با فیلترهای متنوع
- گزارش‌گیری آماری و تحلیلی
- نمودارهای متنوع برای نمایش داده‌ها
- امکان صدور گزارش‌ها به فرمت‌های Excel و PDF
- رابط کاربری چند زبانه (فارسی و انگلیسی)
- داشبورد مدیریتی با نمایش آمار و اطلاعات کلیدی
- پشتیبان‌گیری و بازیابی اطلاعات
- تنظیمات متنوع برای شخصی‌سازی برنامه

## نیازمندی‌ها

- Python 3.6 یا بالاتر
- کتابخانه‌های پایتون مورد نیاز (لیست شده در فایل `requirements.txt`)
- کامپایلر C برای هسته اصلی برنامه
- سیستم عامل ویندوز، لینوکس یا مک

## نصب

### پیش‌نیازها

ابتدا باید کامپایلر C و پایتون 3.6 یا بالاتر را بر روی سیستم خود نصب کنید.

### نصب کتابخانه‌های پایتون

برای نصب کتابخانه‌های مورد نیاز، دستور زیر را اجرا کنید:

```
pip install -r requirements.txt
```

### کامپایل هسته C

برای کامپایل هسته C برنامه، به دایرکتوری اصلی پروژه رفته و دستور زیر را اجرا کنید:

```
cd ../
mkdir build
cd build
cmake ..
make
```

در ویندوز، می‌توانید از Visual Studio یا MinGW برای کامپایل استفاده کنید.

## راه‌اندازی برنامه

برای اجرای برنامه، دستور زیر را از دایرکتوری `python` اجرا کنید:

```
python entry_point.py
```

برای مشاهده گزینه‌های اجرایی برنامه، از دستور زیر استفاده کنید:

```
python entry_point.py --help
```

## استفاده از برنامه

### ورود به سیستم

در اولین اجرای برنامه، باید یک حساب کاربری مدیر ایجاد کنید. برای این کار، اطلاعات درخواستی را در صفحه ثبت‌نام وارد کنید.

پس از ایجاد حساب کاربری، می‌توانید با نام کاربری و رمز عبور خود وارد سیستم شوید.

### ثبت ملک جدید

برای ثبت ملک جدید، از منوی اصلی گزینه مناسب (ملک مسکونی، تجاری یا زمین) را انتخاب کرده و اطلاعات مورد نیاز را وارد کنید.

### جستجوی ملک

برای جستجوی ملک، به تب جستجو رفته و فیلترهای مورد نظر خود را تنظیم کنید.

### گزارش‌گیری

برای ایجاد گزارش‌های مختلف، به تب گزارش‌ها رفته و نوع گزارش، بازه زمانی و سایر تنظیمات مورد نظر را انتخاب کنید.

## ساختار دایرکتوری پروژه

- `src/`: کد منبع هسته C برنامه
- `python/`: کد پایتون رابط کاربری و ارتباط با هسته C
  - `property_management/`: پکیج اصلی پایتون
    - `__init__.py`: فایل آغازین پکیج
    - `app.py`: برنامه اصلی
    - `residential_bridge.py`: پل ارتباطی با بخش املاک مسکونی
    - `commercial_bridge.py`: پل ارتباطی با بخش املاک تجاری
    - `land_bridge.py`: پل ارتباطی با بخش زمین
    - `report_generator.py`: تولیدکننده گزارش‌ها
    - `charts.py`: مدیریت نمودارها
    - `export.py`: صدور داده‌ها
    - `ui/`: ماژول‌های رابط کاربری
- `include/`: هدرهای C
- `data/`: فایل‌های داده
- `docs/`: مستندات
- `tests/`: تست‌ها
