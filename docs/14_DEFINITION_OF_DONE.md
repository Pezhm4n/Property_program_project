# DEFINITION OF DONE (Phase 0.5)

سند **Definition of Done (DoD)** مجموعه‌ای جامع از معیارهایی است که مشخص می‌کند یک Task (یا Feature/Bugfix) چه زمانی واقعاً "تمام شده" و آماده ادغام (Merge) در شاخه اصلی (`main`) محسوب می‌شود.
بدون رعایت تمام شروط زیر، هیچ کاری در سیستم مدیریت املاک به پایان نرسیده است.

## چک‌لیست پایان کار (DoD Checklist)

### 1. عملکرد (Functionality)
- [ ] فیچر یا تسک دقیقاً مطابق با نیازمندی‌ها (Requirements) در بستر محلی (Local) به درستی کار می‌کند.
- [ ] هیچ باگِ بحرانیِ شناخته‌شده‌ای پیرامون ویژگی جدید وجود ندارد.

### 2. بیلد و کامپایل (Build & Compilation)
- [ ] کامپایل کد C روی پلتفرم هدف با موفقیت و **بدون هیچ Warningای** (با فلگ‌های سخت‌گیرانه `/W4` یا `-Wall -Wextra -Werror`) انجام می‌شود.
- [ ] کدهای پایتون توسط لینترهای **Ruff** و فرمتر **Black** بررسی شده و هیچ خطای ساختاری یا فرمتی وجود ندارد.

### 3. تست‌ها (Testing)
- [ ] تمام تست‌های قبلی (Regression) در سرور CI/CD سبز هستند.
- [ ] برای فیچر یا تابع جدید، Unit Testهای مجزا در سمت C و Python نوشته شده‌اند و Pass شده‌اند.
- [ ] در صورت ارتباط با پایگاه داده، تست‌های Integration روی پایگاه داده موقت انجام شده‌اند.
- [ ] در صورت ساخت UI، رفتار صحیح با `pytest-qt` صحه‌گذاری شده است.
- [ ] نشت حافظه (Memory Leak) برای خروجی‌های تابع DLL در مرز پایتون کنترل شده و `re_free_string` فراخوانی شده است.

### 4. مستندات (Documentation)
- [ ] هدرهای پابلیک C با استاندارد Doxygen مستند شده‌اند.
- [ ] در صورت تغییر در فیلدهای JSON یا ساختار جداول دیتابیس، فایل‌های Markdown فازهای 0 (نظیر `06_DATABASE_DESIGN.md` یا `08_JSON_SCHEMA.md`) بلافاصله به روزرسانی شده‌اند.
- [ ] اگر تصمیم معماری جدیدی گرفته شده است، یک ADR جدید ثبت شده است.

### 5. بازبینی کد (Code Review)
- [ ] کد توسط حداقل یک توسعه‌دهنده دیگر به طور کامل بازبینی (Review) و تایید (Approved) شده است.
- [ ] هیچ‌گونه کدهای کامنت‌شده‌ی زائد (Dead Code) در کامیت نهایی وجود ندارد.
- [ ] کدهای محرمانه یا نام کاربری و پسورد هاردکد شده (Hardcoded Secrets) در متن کد نیست.

### 6. یکپارچگی CI/CD
- [ ] تمامی Checkهای تعریف شده در GitHub Actions پاس شده‌اند (سبز شده‌اند).

**نتیجه:** تنها زمانی که تمامی این ۶ بند تیک بخورند، تسک به عنوان `Done` نشان‌گذاری می‌شود.
